Sobre el proyecto

Una pieza de portafolio pensada para mostrar criterio tecnico y visual

Sabor Valle es una pagina conceptual inspirada en la gastronomia del Valle del Cauca. Nacio como una idea sencilla y fue creciendo hasta convertirse en una demo completa con secciones visuales, categorias de menu, reservas y despliegue online.

El objetivo no fue solo construir una pagina bonita, sino tambien demostrar capacidad para corregir errores, estructurar un proyecto, integrar un backend funcional y tomar decisiones de producto usando IA como apoyo de desarrollo.

Objetivo

Diseñar una experiencia web atractiva que muestre frontend, backend y despliegue en un solo proyecto.

Enfoque

Mostrar proceso real: iteracion visual, integracion de rutas, contenido personalizado y mejoras guiadas por objetivos.

Stack del proyecto

Tecnologias y habilidades demostradas

Frontend

HTML estructurado, Tailwind CSS para maquetacion visual y JavaScript para interaccion del formulario.

Backend

Python con FastAPI para recibir reservas, servir el frontend y mantener una estructura funcional de servidor.

Deploy

Proyecto preparado para GitHub y Render, demostrando publicacion, estructura limpia y despliegue de una demo real.
